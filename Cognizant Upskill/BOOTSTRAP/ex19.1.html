Steps to set up Bootstrap 5 with Sass using npm:
1. npm init -y
2. npm install bootstrap sass
3. Create custom SCSS file (e.g., custom.scss):
   @import "node_modules/bootstrap/scss/bootstrap";
4. Compile: sass custom.scss custom.css
5. Link custom.css in HTML.

Project structure example includes node_modules/, custom.scss, and compiled CSS.
