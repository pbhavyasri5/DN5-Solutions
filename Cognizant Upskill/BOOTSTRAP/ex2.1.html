Bootstrap directory structure (example):
bootstrap-5.3.0-dist/
├── css/
│   ├── bootstrap.css
│   ├── bootstrap.min.css
│   └── bootstrap.rtl.min.css
├── js/
│   ├── bootstrap.bundle.js
│   └── bootstrap.bundle.min.js
└── icons/ (if downloaded separately)
    ├── bootstrap-icons.css
    └── fonts/

Purpose:
- css/: contains compiled Bootstrap styles, including minified and RTL versions.
- js/: contains Bootstrap JavaScript plugins, bundled with Popper.
- icons/: contains Bootstrap Icons CSS and font files for scalable vector icons.
